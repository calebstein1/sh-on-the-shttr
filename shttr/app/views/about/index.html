About

Shell on the Shttr was created by Caleb Stein, originally as a joke; then as way for me to streamline development of my own personal sites; and finally as a real, usable product (maintaining the joke name, of course).

It is a cutting-edge, databaseless backend web framework built using shell scripts.

A default Shttr installation comes configured with Turbo as a frontend framework and Pico.css as a CSS framework. This configuration allows for visually appealing and responsive apps to be rapidly built on the Shttr backend, though you're free of course to swap out whatever components you want.
